Forme réelle du circuit

Ce que montre ce graphique : les positions GPS des deux tours. Comment le lire : blanc = référence, bleu = comparé, jaune = départ. Ce que nous observons : uniquement les écarts mesurés. Delta temps

Écart cumulé sur la distance

Delta cumulé — secondes

Lecture : vert = gain, rouge = perte, bleu = écart faible. Le calcul est aligné sur la progression du tour.

Commandes pilote

Courbes synchronisées sur la distance

Accélérateur — %

Lecture : une valeur élevée correspond à davantage d'accélérateur. Une absence de courbe signifie que le capteur n'est pas disponible.

Frein — %

Lecture : permet de comparer le début, la durée et l'intensité du freinage lorsque le canal existe.

Volant — valeur capteur

Lecture : les variations représentent les mouvements du volant. La calibration en degrés n'est pas inventée.

Régime moteur — tr/min

Lecture : le régime est comparé sur la même distance que la vitesse et le delta.

Température — °C

Lecture : la température n'est affichée que si elle est réellement présente dans l'archive.
